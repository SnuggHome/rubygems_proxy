===============================================
==              RUBYGEMS MIRROR              ==
===============================================

Add the following line to your Gemfile:

  source "http://gems.simplesideias.com.br"

And this is my ~/.gemrc file:

  ---
  :backtrace: false
  :benchmark: false
  :bulk_threshold: 1000
  :sources:
  - http://gems.simplesideias.com.br/
  :update_sources: true
  :verbose: true
  gem: --no-rdoc --no-ri
